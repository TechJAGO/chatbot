Possible issues related to Credentials

  • Credentials might be wrong, try to re-add the credentials or make a new profile
  • If it is a Windows device, login to end device and then do the following:
    Run -> Wbemtest -> Enter the required details.
    If you are not able to connect then it is a device end issue and not OPM issue.
  • For a network device, from OPM UI go to
    Setting -> Tools -> MIB Browser -> Enter the device name , credentials and from the left coloumn select any one from the list and click on "GET". If the credentials are correct then you will be able to get the info otherwise it will throw an error.
